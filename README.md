# Market Entry Calculator

A web-based financial calculator to help businesses evaluate whether to enter a new market.

## What It Does

This calculator helps you answer: **"Should we enter this market, at what price, and with what expected return?"**

Enter your market assumptions, pricing, and costs to get:
- 3-year revenue projections
- Break-even analysis
- ROI calculations
- GO/NO-GO/REVIEW recommendations

## Features

- ✅ Real-time calculations as you type
- ✅ Interactive inputs for market size, pricing, and costs
- ✅ Visual decision indicators (color-coded)
- ✅ Mobile-friendly responsive design
- ✅ No data stored - complete privacy

## Tech Stack

- React 18
- Tailwind CSS
- Lucide React Icons

## Getting Started

To run locally:

```bash
npm install
npm start
```

Opens on `http://localhost:3000`
